Git

  • Enter your Git notes here
  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git add -A: add the changed files in the working directory to the staging area of Git.
  • git commit -m "message": commit changes to the repo and adds a message
  • git pull origin main: syncs the local branch with the remote branch in GitHub.
  • It is a good habit to always pull the base branch into your feature brach before pushing up any code
  • git push origin feature/add-html: Push all of the changes
